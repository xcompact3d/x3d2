io_manager_t

Provides a high-level manager that orchestrates all checkpoint, snapshot, and statistics operations.

Its purpose is to simplify the main simulation loop by providing a single point of contact for all I/O-related actions. The mainprogram only needs to interact with the io_manager_t type, which then delegates tasks to the specialised checkpoint, snapshot, and statistics managers.
